# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行裂缝检测
- 提供完整的源码、训练好的模型及4000张标注数据集
- 支持选择检测模型，自定义置信度和IOU阈值
- 可选择显示标签和原图
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存功能
- 提供项目运行教程和完整的程序文件

## 适用场景
- 建筑物表面裂缝检测
- 道路路面状况评估
- 桥梁结构安全监测
- 工业设备表面缺陷检测
- 科研研究与教育演示

## 技术特点
- 基于YOLOv8的实例分割技术，专注于裂缝检测
- 使用PyQt5构建图形用户界面，操作简便
- 集成OpenCV进行图像处理和显示
- 支持多种输入源（图片、视频、摄像头）
- 提供完整的环境配置方案（Python3.9、PyTorch1.9等）
